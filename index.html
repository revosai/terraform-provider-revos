Terraform Provider Revos

This is a Terraform provider network mirror for revosai/revos.

Installation

Add to ~/.terraformrc:

provider_installation {
  network_mirror {
    url = "https://revosai.github.io/terraform-provider-revos/"
    include = ["revosai/revos"]
  }
  direct {
    exclude = ["revosai/revos"]
  }
}

Then use in your Terraform config:

terraform {
  required_providers {
    revos = {
      source  = "revosai/revos"
      version = "0.1.0"
    }
  }
}
